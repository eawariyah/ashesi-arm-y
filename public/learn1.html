Embedded Systems Essentials with Arm: Getting Started

Are you interested in improving your skills in embedded systems and earning certificates? Look no further than the course below!

About this course

This course will teach you the fundamentals of embedded systems and why Arm-based processors are particularly well-suited for IoT applications. You will learn how to implement real-world embedded system projects on a simulated microcontroller using the Mbed API. The course includes instructions on how to access an Mbed simulator, so you can apply your new knowledge and skills to prototype and build real-world embedded applications quickly, without the trouble or expense of sourcing hardware.

Embedded systems are everywhere, from mobile devices to the transformative AI and 5G-powered IoT of the near future. As we move towards trillions of intelligent, connected devices, itâ€™s crucial that the next generation of engineers have a thorough understanding of how to rapidly build and deploy a modern embedded system solution.

Over the six modules, you will gain a key overview of:

  1. Embedded systems and their importance in modern computing applications
  2. Industry-standard Arm-based microcontroller architecture, their features, and programming models

What you'll learn

By taking this course, you will learn how to:

  1. Modify skeleton code to implement functionality for multiple applications: an audio wave generator, an audio player, and controlling peripherals using analog and digital I/O interfaces, interrupts, timers, and pulse width modulation.
  2. Understand applications of embedded systems and how a modern API can help accelerate IoT deployment.
  3. Implement real-world embedded system projects on a simulated microcontroller using the Mbed API.
  4. Learn key features of Arm processors and architectures as the basis of modern embedded computing.

Syllabus

The course consists of six modules:

  1. Introduction to embedded system
  2. Introduction to the Mbed Platform and CMSIS
  3. Digital I/O
  4. Interrupts and Low Power Features
  5. Analog I/O
  6. Timer and Pulse-width Modulation
